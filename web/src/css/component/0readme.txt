代码规范：

1. con 打头代表布局相关
	容器内的元素
	布局的级别
	
	con			最外层容器
	wrapper		为了一些目的制作的包裹器
	
	top			顶部
	banner		运营图片
	main		主体内容
	foot		尾部
	nav			导航
	
2. 组件样式
	btn			按钮
	label		标签
	table		表格  
	list		列表   水平列表  垂直列表
	
	img			图像
	icon		小图标
	
	input		输入框   文本  多行   单选   多选   
	slt  		值列表选取
	
	
3. 大组件
	dlg  		对话框
	calendar	日历
	tree
